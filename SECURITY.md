# Security Policy

## Reporting a Vulnerability

如果你发现了安全漏洞，请**不要**公开提交 Issue。

请通过以下方式私密报告：

- 发送邮件至仓库所有者的 GitHub 关联邮箱
- 或通过 GitHub 的 [Security Advisories](https://github.com/IOAFukua/GamePulseMonitor/security/advisories) 功能提交

我们会尽快确认并修复。

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :white_check_mark: |
